Project Brief

"The definitive summary of the 4simpleproblems project."

Overview

4simpleproblems (4SP) is a student-made web platform designed for bored school days. It blends utilities, multimedia tools, and soundboards into a tool that feels fast, personal, and strictly non-corporate. What began as a joke page has evolved into a robust, dashboard-style platform with advanced local security features.

Core Philosophy

  • Feel personal, not "fake-professional."
  • Zero external dependencies for the local version.
  • Fast load times, hard to block.
  • Provide entertainment without tracking.

Tech Stack

  • HTML5, CSS3, Vanilla JavaScript.
  • Firebase (Auth/Functions) for online tiers.
  • Tailwind CSS via CDN.
  • Client-side focus for maximum privacy.
