The Rapid Evolution of 4SP

Creation and Vision

4SP was founded on March 13, 2025, starting as a simple two-page website. The initial vision was to provide a basic soundboard and sound request system for students.

The Turning Point (V4)

Version 4, released in August 2025, represented a major shift. It introduced the modern dashboard layout, widgets, and essential QOL features like Tab Disguising and the Panic Key, creating a more structured platform.

The Professional Rebuild (V5)

After a temporary platform block in October 2025, Version 5 was created in just two months. It is a complete rebuild, focusing on security, professional design, and comprehensive student utility.

Major Version Highlights

  1. Version 5 (Dec 2, 2025)

    The Professional Rebuild - Full Production

    Complete rewrite with a focus on security, professional design, and comprehensive student utility. Introduced the customizable navigation bar and the Games Collection (1,600+ games). Rapidly developed in two months after Version 4 was blocked.

  2. Version 4 (Aug 22, 2025)

    The Dashboard Launch

    Introduced the dark grey/purple scheme, a structured sidebar/dashboard, widgets, Tab Disguising, and the Panic Key. Proxies and other experimental pages were removed.

  3. Version 3 (May 15, 2025)

    The Visual Redesign

    Shifted from a black-and-white to a light white-and-grey palette using the Outfit font, with minimal functional updates.

  4. Version 2 (Apr 11, 2025)

    The Great Expansion

    Expanded significantly to 36 pages, introducing games, playlists, proxies, and multimedia. The repository became cluttered, and a pirated movie was later removed.

  5. Version 1 (Mar 13, 2025)

    The Genesis

    Launched as a simple two-page website featuring a soundboard (22 normal, 4 explicit sounds) and a request page.

Version 5: Feature Highlights

Security & Access

Professional User Management

Version 5 focuses on providing a secure and flexible authentication experience using modern cloud infrastructure (Firebase).

  • Multi-Provider Sign-In:Supports Google, Microsoft, GitHub, and X (Twitter).
  • Customizable Navbar & Pin Button:Reduced page clutter and offers quick access to favorite apps.
  • Adaptive UI:Optimized for both desktop and mobile use.
Entertainment & Data

Comprehensive Games Collection

Version 5 aggregates a massive library for student downtime, now featuring game data management for a persistent experience.

  • Game Data Management:Allows for saving progress across sessions.
  • Games Library:Over 1,600 games, aggregated from sources like GN-Math and StrongdogXP.
  • Retro Games:Includes HTML-based classics like Eaglercraft, Grand Theft Auto (original), Doom, and SimCity 1 & 2000.
