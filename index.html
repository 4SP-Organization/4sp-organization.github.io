The Rapid Evolution of 4SP

Creation and Vision

4SP was founded on March 13, 2025, starting as a simple two-page website. The initial vision was to provide a basic soundboard and sound request system for students.

The Turning Point (V4)

Version 4, released in August 2025, represented a major shift. It introduced the modern dashboard layout, widgets, and essential QOL features like Tab Disguising and the Panic Key.

The Professional Rebuild (V5)

After a platform block in October 2025, Version 5 was created. It is a complete rebuild, focusing on security, professional design, and comprehensive student utility.

Project Brief

"This is the biggest, most complete summary of the 4simpleproblems project."

Overview

4simpleproblems, usually called 4SP, is a student-made web project built for bored school days. It mixes utilities, games, soundboards, and small tools that feel fast, local, and fun instead of polished or corporate. The project evolved from a couple of joke pages into a full dashboard-style platform with accounts, cloud functions, and serious security decisions.

Core Idea

  • Make something that feels personal and not "fake-professional."
  • Avoid corporate aesthetics.
  • Ensure it is fast to load and hard to block.
  • Provide entertainment when school is boring.

Tech Stack

  • Standard HTML, CSS, and JavaScript.
  • Firebase: Hosting, Firestore, Cloud Functions, and Auth.
  • Heavy use of CDNs, specifically jsDelivr.
  • Minimal backend exposure to protect the client-side focus.

Version History

Version 1: The Genesis

Launched March 13, 2025. Featured a 20-sound soundboard, an autoclicker, and a request page. No accounts or real structure.

Version 2: Expansion

Expanded to 36 pages including media, playlists, proxies, and accounts. Firebase usage increased significantly.

Version 3: Visual Reskin

First big identity shift. Introduced a white layout with sharp corners and cleaner structures while keeping the chaotic fun.

Version 4: The Platform

Major turning point. Introduced the dashboard layout, Koyeb-inspired widgets (Weather, Notes, Calculator), and stealth features like the Panic Key and Tab Disguise.

Version 5: Project Zirconium

The most professional version. Vercel-inspired design, dynamic navigation, and the introduction of the unblockable Downloadable Version (DV) client.

Final Thoughts

4SP is a long-running experiment and personal sandbox. It is a response to restrictions and a place where small tools matter. Every time someone tries to stop it, the project grows. It has history, users, and scars from blocks—and it is still changing.
