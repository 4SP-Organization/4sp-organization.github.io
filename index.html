The Data

Project Brief & Timeline

4simpleproblems (4SP) is a student-developed web platform designed for use in restricted environments. It has grown from a simple webpage into a distributed software toolkit.

Core Strategy

Decentralization and the use of official repositories (npm) to maintain uptime and availability.

Technical Stack

Evolved from basic iframes in V2 to a complex client-side application in V5 MAX.

Global Footprint
